Exercício Séries Reps Carga (kg) Sets
Dica: marque cada set conforme conclui. Use o botão de descanso entre as séries.

Como usar

  1. Escolha a data e o treino (A, B, C, D ou Descanso).
  2. Clique em Carregar/Resetar treino do dia para montar a lista de exercícios.
  3. Preencha carga (kg) se desejar e marque os sets conforme concluir.
  4. Ajuste o descanso (segundos) e use o cronômetro entre as séries.
  5. Escreva observações e clique em Salvar. Tudo fica no seu navegador.
  6. Use Exportar JSON para baixar um arquivo com seu histórico.
