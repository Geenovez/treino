Como usar (PWA)

  1. Abra este site no celular pelo GitHub Pages.
  2. No Android (Chrome): toque em ⋮ > Adicionar à tela inicial. No iOS (Safari): toque em Compartilhar > Adicionar à Tela de Início.
  3. Depois do primeiro acesso, funciona offline. Seus dados ficam no aparelho.
